# Data Interpretation

When we are working with data in form of bytes, we often need to interpret this data in a specific way.
Two bytes can represent a short, four bytes can represent an integer, and so on.

## Source Types

Primitives extends the functionality of the following types to allow for easy interpretation of data:

- `ByteArray`
- `Array<Byte>`
- `List<Byte>` >> _`MutableList<Byte>`_

All operations are available for all three types.

## Operations

### to

The to function converts the data to a type. This function needs the Array/ByteArray/List to be of the correct size.

```kt
import com.shakelang.util.primitives.bytes.toShort

val bytes = byteArrayOf(0x01, 0x02)
val short = bytes.toShort()
```

[Works with all target types](#target-types)

<details>
  <summary>See all functions</summary>

- `toByte()` - Converts the data to a `Byte`: `[0x01]` >> `0x01`
- `toShortBE()` - Converts the data to a `Short` _(big endian)_: `[0x01, 0x02]` >> `0x0102`
- `toShortLE()` - Converts the data to a `Short` _(little endian)_: `[0x01, 0x02]` >> `0x0201`
- `toShort()` - Converts the data to a `Short` _(big endian)_: `[0x01, 0x02]` >> `0x0102`
- `toIntBE()` - Converts the data to a `Int` _(big endian)_: `[0x01, 0x02, 0x03, 0x04]` >> `0x01020304`
- `toIntLE()` - Converts the data to a `Int` _(little endian)_: `[0x01, 0x02, 0x03, 0x04]` >> `0x04030201`
- `toInt()` - Converts the data to a `Int` _(big endian)_: `[0x01, 0x02, 0x03, 0x04]` >> `0x01020304`
- `toLongBE()` - Converts the data to a `Long` _(big endian)_: `[0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07, 0x08]` >> `0x0102030405060708`
- `toLongLE()` - Converts the data to a `Long` _(little endian)_: `[0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07, 0x08]` >> `0x0807060504030201`
- `toLong()` - Converts the data to a `Long` _(big endian)_: `[0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07, 0x08]` >> `0x0102030405060708`
- `toFloatBE()` - Converts the data to a `Float` _(big endian)_: `[0x3F, 0x80, 0x00, 0x00]` >> `1.0`
- `toFloatLE()` - Converts the data to a `Float` _(little endian)_: `[0x00, 0x00, 0x80, 0x3F]` >> `1.0`
- `toFloat()` - Converts the data to a `Float` _(big endian)_: `[0x3F, 0x80, 0x00, 0x00]` >> `1.0`
- `toDoubleBE()` - Converts the data to a `Double` _(big endian)_: `[0x3F, 0xF0, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00]` >> `1.0`
- `toDoubleLE()` - Converts the data to a `Double` _(little endian)_: `[0x00, 0x00, 0x00, 0x00, 0x00, 0x00, 0xF0, 0x3F]` >> `1.0`
- `toDouble()` - Converts the data to a `Double` _(big endian)_: `[0x3F, 0xF0, 0x00, 0x00, 0x00, 0x00, 0x00, 0x00]` >> `1.0`
- `toUByte()` - Converts the data to a `UByte`: `[0x01]` >> `0x01`
- `toUShortBE()` - Converts the data to a `UShort` _(big endian)_: `[0x01, 0x02]` >> `0x0102`
- `toUShortLE()` - Converts the data to a `UShort` _(little endian)_: `[0x01, 0x02]` >> `0x0201`
- `toUShort()` - Converts the data to a `UShort` _(big endian)_: `[0x01, 0x02]` >> `0x0102`
- `toUIntBE()` - Converts the data to a `UInt` _(big endian)_: `[0x01, 0x02, 0x03, 0x04]` >> `0x01020304`
- `toUIntLE()` - Converts the data to a `UInt` _(little endian)_: `[0x01, 0x02, 0x03, 0x04]` >> `0x04030201`
- `toUInt()` - Converts the data to a `UInt` _(big endian)_: `[0x01, 0x02, 0x03, 0x04]` >> `0x01020304`
- `toULongBE()` - Converts the data to a `ULong` _(big endian)_: `[0x01, 0x02, 0x03, 0x04, 0x05, 0x06, 0x07, 0x08]` >> `0x0102030405060708`

### get

The get function gets a value from the data at a specific index.

```kt
import com.shakelang.util.primitives.bytes.getShort

val bytes = byteArrayOf(0x01, 0x02)
val short = bytes.getShort(0)
```

```kt
import com.shakelang.util.primitives.bytes.getShort

val bytes = byteArrayOf(0x01, 0x02, 0x03, 0x04)

val short1 = bytes.getShort(1) // >> 0x0203
val short2 = bytes.getShort(0) // >> 0x0102
```

### set

The set function sets a value at a specific index.
This function is not available on `List<Byte>` as it is immutable, but it is available on `MutableList<Byte>`.

```kt
import com.shakelang.util.primitives.bytes.setShort

val bytes = byteArrayOf(0x01, 0x02)
bytes.setShort(0, 0x0304)
```

```kt
import com.shakelang.util.primitives.bytes.setShort

val bytes = byteArrayOf(0x01, 0x02, 0x03, 0x04)

bytes.setShort(1, 0x0506) // >> [0x01, 0x05, 0x06, 0x04]
```

### append

This only works on `MutableList<Byte>` (array size is fixed, `List<Byte>` is immutable).

The append function appends a value to the end of the list.

```kt
import com.shakelang.util.primitives.bytes.appendShort

val bytes = mutableListOf<Byte>(0x01, 0x02)
bytes.appendShort(0x0304) // >> [0x01, 0x02, 0x03, 0x04]
```
