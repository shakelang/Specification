# CommonIO

CommonIO is a kmp common streaming library that provides a common interface for reading and writing data
from different sources. It is designed to be used in multiplatform projects for writing one codebase that
can be used on all platforms.

## Installation

### Gradle

`build.gradle`

```groovy
implementation "com.shakelang.util:common-io:$version"
```

`build.gradle.kts`

```kotlin
implementation("com.shakelang.util:common-io:$version")
```

### Maven

```xml
<dependency>
  <groupId>com.shakelang.util</groupId>
  <artifactId>common-io</artifactId>
  <version>$version</version>
</dependency>
```
